# COVID-19 Spread Prediction Dashboard

A comprehensive Streamlit dashboard for visualizing and analyzing COVID-19 infection rate predictions using integrated modeling approaches.

## Features

- **Spatial Analysis**: Interactive maps showing true vs predicted infection rates
- **Performance Metrics**: Detailed model performance analysis
- **Feature Analysis**: Explore relationships between demographic factors and infection rates
- **Bias Analysis**: Identify and visualize prediction biases
- **Uncertainty Analysis**: Understand model confidence across different areas
